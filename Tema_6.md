Дана строка в виде случайной последовательности чисел от 0 до 9 (длина строки минимум 15 символов). Требуется создать словарь, который в качестве ключей будет принимать данные числа (т. е. ключи будут типом int), а в качестве значений – количество этих чисел в имеющейся последовательности. Для построения словаря создайте функцию, принимающую строку из цифр. Функция должна возвратить словарь из 3-х самых часто встречаемых чисел, также эти значения нужно вывести в порядке возрастания ключа.

```python
def count_digits(sequence):
    frequency = {}
    # проходим по каждому символу в строке
    for char in sequence:
        digit = int(char)
        if digit in frequency:
            frequency[digit] += 1
        else:
            frequency[digit] = 1
    # сортируем элементы словаря по убыванию значений
    sorted_frequency = sorted(frequency.items(), key=lambda item: item[1], reverse=True)
    # извлекаем три самых частых элемента
    top_three = sorted_frequency[:3]
    # сортируем их по возрастанию
    result = dict(sorted(top_three))
    return result
digit_sequence = "7469844807348909647459856790"
result = count_digits(digit_sequence)
print(result)

```

- Создание словаря `frequency:` который используется для подсчета входа каждой цифры.
- Проход по символам строки осуществляется так: для каждной цифры в строке: преобразуем цифру в целое число. Если цифра уже в словаре, увеличиваем её значение на 1. Если нет, добавляем её в словарь с начальным значением 1.

Ваш хороший друг владеет офисом со входом по электронным картам, ему нужно чтобы вы написали программу, которая показывала в каком порядке сотрудники входили и выходили из офиса. Определение сотрудника происходит по id. Напишите функцию, которая на вход принимает кортеж и случайный элемент (id), его можно придумать самостоятельно. Требуется вернуть новый кортеж, начинающийся с первого появления элемента в нем и заканчивающийся вторым его появлением включительно.
Если элемента нет вовсе – вернуть пустой кортеж.
Если элемент встречается только один раз, то вернуть кортеж, который начинается с него и идет до конца исходного.
Входные данные:
(1, 2, 3), 8)
(1, 8, 3, 4, 8, 8, 9, 2), 8)
(1, 2, 8, 5, 1, 2, 9), 8)
Ожидаемый результат:
()
(8, 3, 4, 8)
(8, 5, 1, 2, 9)

```python
def find_employee_range(record, employee_id):
    first_index = -1
    second_index = -1
    for index, value in enumerate(record):
        if value == employee_id:
            if first_index == -1:
                first_index = index  # сохраняем индекс первого вхождения
            elif second_index == -1:
                second_index = index 
                break 
    # если элемент не найден
    if first_index == -1:
        return ()
    if second_index == -1:
        return record[first_index:]  
    return record[first_index:second_index + 1]  
print(find_employee_range((1, 2, 3), 8))  
print(find_employee_range((1, 8, 3, 4, 8, 8, 9, 2), 8))
print(find_employee_range((1, 2, 8, 5, 1, 2, 9), 8)) 

```

- Поиск индексов: перебираем кортеж с помощью enumerate, чтобы получить и индекс, и значение. Для первого вхождения сохраняем `first_index`, для второго в `second_index`.

- Словарь в Python — это неупорядоченная коллекция пар ключ-значение, где каждому ключу соответствует значение. Это структура данных, в которой каждый ключ должен быть уникален. Создать словарь можно с помощью фигурных скобок {} или используя функцию dict(). Чтобы получить значение, нужно указать соответствующий ключ.  Словари можно изменять. Можно добавлять, изменять или удалять пары "ключ-значение". Словари идеально подходят для хранения данных, доступ к которым нужно организовать по ключу, например, в случаях, когда нужно хранить данные о пользователях или других объектах.

- Кортежи в Python — это неизменяемые последовательности данных. Они могут хранить элементы различных типов. Кортежи создаются с использованием круглых скобок (). После создания кортежа его содержимое нельзя изменить (добавить, удалить или изменить элементы). Кортежи могут содержать другие кортежи, списки или даже словари. Кортежи используются, когда нужно хранить несколько значений в одной переменной и гарантировать, что данные не будут изменены.
