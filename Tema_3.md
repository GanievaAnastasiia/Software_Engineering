## Лабораторная работа №1
### Создайте две переменные, значение которых будете вводить через консоль. Также составьте условие, в котором созданные ранее переменные будут сравниваться, если условие выполняется, то выведете в консоль «Выполняется», если нет, то «Не выполняется».

```python
one = int(input('Введите значение первой переменной: '))
two = int(input('Введите значение второй переменной: '))
if one >= two:
    print('Выполняется')
else:
    print('Не выполняется')
```
### Результат.

![Меню](https://github.com/GanievaAnastasiia/Software_Engineering/blob/Тема_3/images3/1.png)

## Выводы

Код запрашивает у пользователя ввод двух целых чисел и сравнивает их. В зависимости от сравнения, он выводит одно из двух сообщений. 
Программа будет выводить "Выполняется", если первое число больше или равно второму, и "Не выполняется" в противном случае.
`if one >= two:` выполняется, если первая переменная больше или равна второй переменной. В иных случаях будет выполнятся `else`.

## Лабораторная работа №2
### Напишите программу, которая будет определять значения переменной меньше 0, больше 0 и меньше 10 или больше 10. Это нужно реализовать при помощи одной переменной, значение которой будет вводится через консоль, а также при помощи конструкций if, elif, else.

```python
one = int(input('Введите значение переменной: '))
if one < 0:
    print('Меньше 0')
elif 0 < one < 10:
    print('Больше 0 и меньше 10')
else:
    print('Больше 10')
```

![Меню](https://github.com/GanievaAnastasiia/Software_Engineering/blob/Тема_3/images3/2.png)

Оператор if-elif-else проверяет несколько условий.
Если `if one < 0:` истинно, выполняется ` print('Меньше 0')`.
Если `if one < 0:` ложно, то проверятеся  условие `elif 0 < one < 10:`.
Если оба условия ложны, выполняется `else`.

## Лабораторная работа №4
### Напишите программу, которая будет определять находится ли переменная в указанном массиве и если да, то проверьте четная она или нет. Самостоятельно протестируйте данную программу с разными значениями переменной value.

```python
numbers = [12,525,88,1,29,114,289,2822]
value = int(input('Введите значение переменной: '))
if value in numbers:
    if value % 2 == 0:
        print('Переменная четная и есть в массиве numbers')
    else:
        print('Переменная нечетная и есть в массиве numbers')
else:
    print(f"Переменной нет в массиве numbers и она равна {value}")
```

### Результат

![Меню](https://github.com/GanievaAnastasiia/Software_Engineering/blob/Тема_3/images3/4.png)

### Выводы

В коде происходит проверка, есть ли введенное пользователем значение в списке numbers, и является ли это значение четным или нечетным.
Список `numbers`: содержит заранее определенные значения.
Пользователь вводит значение, которое затем преобразуется в целое число с помощью int().
Первый оператор if: проверяет, содержится ли введенное значение value в списке numbers. Если значение найдено, выполняется вложенный оператор if.
Вложенный оператор if: проверяет, является ли значение четным `(value % 2 == 0)`: Если да, выводится сообщение: `'Переменная четная и есть в массиве numbers'`. Если нет, выводится сообщение: 'Переменная нечетная и есть в массиве numbers'.
Оператор else: если введенное значение не найдено в списке, выводится сообщение: `f"Переменной нет в массиве numbers и она равна {value}"`.

## Лабораторная работа №5
### Напишите программу, в которой циклом for значения переменной і будут меняться от 0 до 10 и посмотрите, как разные виды сравнений и операций работают в цикле.

```python
for i in range(10):
    print(f"i = {i}")
    if i == 0:
        i += 2
    if i == 1:
        continue
    if i == 2 or i == 4:
        print('Переменная равна 2 или 4')
    elif i in [5, 7, 9]:
        print('Переменная равна 5, 7 или 9')
    else:
        break
```

### Результат

![Меню](https://github.com/GanievaAnastasiia/Software_Engineering/blob/Тема_3/images3/5.png)

### Выводы

Программа демонстрирует последовательное выполнение условий с помощью операторов continue и break.
`i = 0` — На первой итерации i равно 0, оно выводится на экран.
`i = 1` — На второй итерации i равно 1, оно также выводится, но затем срабатывает оператор continue, и вывод на этом этапе заканчивается для этой итерации.
`i = 2` — На третьей итерации i равно 2, выводится значение i, и сразу затем срабатывает условие, и выводится сообщение 'Переменная равна 2 или 4'.
`i = 3` — На четвертой итерации i равно 3, оно выводится, но не удовлетворяет условиям, и срабатывает else, что приводит к завершению цикла.

## Лабораторная работа №6
### Напишите программу, в которой при помощи цикла for определяется есть ли переменная value в строке string и посмотрите, как работает оператор else для циклов. Самостоятельно посмотрите, что выведет программа, если значение переменной value оказалось в строке string.

```python
string = 'Привет всем, кто изучает Python'
simb = input('Буква которую вы ищите: ')
for i in string:
    if i == simb:
        index = string.find(simb)
        print(f"Буква '{simb}' есть в строке под {index} индексом")
        break
else:
    print(f"Буквы '{simb}' нет в указанной строке")
```

### Результат

![Меню](https://github.com/GanievaAnastasiia/Software_Engineering/blob/Тема_3/images3/6.png)

### Выводы

Программа сначала получает строку string и букву simb, которую хочет найти пользователь.
Затем она проходит в цикле по каждому символу в строке string. При нахождениисимвола впервые, совпадающим с simb, программа находит индекс этого символа и его самого.
Если цикл завершится без нахождения символа, сработает блок else, и программа сообщит, что буквы в строке нет.
Таким образом, вывод программы зависит введет ли пользователь букву, находящуюся в строке, или нет.
