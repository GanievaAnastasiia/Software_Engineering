## Лабораторная работа №5
### Напишите программу, в которой циклом for значения переменной і будут меняться от 0 до 10 и посмотрите, как разные виды сравнений и операций работают в цикле.

```python
for i in range(10):
    print(f"i = {i}")
    if i == 0:
        i += 2
    if i == 1:
        continue
    if i == 2 or i == 4:
        print('Переменная равна 2 или 4')
    elif i in [5, 7, 9]:
        print('Переменная равна 5, 7 или 9')
    else:
        break
```

### Результат

![Меню](https://github.com/GanievaAnastasiia/Software_Engineering/blob/Тема_3/images3/5.png)

### Выводы

Программа демонстрирует последовательное выполнение условий с помощью операторов continue и break.
`i = 0` — На первой итерации i равно 0, оно выводится на экран.
`i = 1` — На второй итерации i равно 1, оно также выводится, но затем срабатывает оператор continue, и вывод на этом этапе заканчивается для этой итерации.
`i = 2` — На третьей итерации i равно 2, выводится значение i, и сразу затем срабатывает условие, и выводится сообщение 'Переменная равна 2 или 4'.
`i = 3` — На четвертой итерации i равно 3, оно выводится, но не удовлетворяет условиям, и срабатывает else, что приводит к завершению цикла.

### Напишите программу используя цикл while, внутри которого есть какие-либо проверки, но быть осторожным, поскольку циклы while при неправильно написанных условиях могут становится бесконечными, как указано в примере далее.

```python
num = 0
while num < 10:
    num += 5
    if num % 2 == 0:
        print(num)
    else:
        print(num)
        num -= 1
```

### Результат

![Меню](https://github.com/GanievaAnastasiia/Software_Engineering/blob/Тема_3/images3/8.png)

### Выводы

Первое выполнение: когда num = 0, цикл while срабатывает, num увеличивается до 5, это нечетное число, выводится 5, затем num уменьшается до 4.
Второе выполнение: num = 4, оно четное, выводится 4.
Третье выполнение: num снова увеличивается до 9, нечетное, выводится 9, затем num уменьшается до 8.
Четвертое выполнение: num = 8, четное, оно выводится.
Пятое выполнение: num снова увеличивается на 5, становится 13, что больше 10, и цикл завершается.
Код выводит на экран 5, 4, 9, и 8. Это происходит через последовательные итерации, где числа меняются в зависимости от четности.
Контролирование переменной num, чтобы она оставалась меньше 10, но из-за уменьшения при нечетных значениях, она может изменяться, что добавляет элемент непредсказуемости.
Код будет выводить числа в интервале от 5 до 9 и завершит работу, когда num станет равным или больше 10.

## Лабораторная работа №9
### Напишите программу с использованием вложенных циклов и одной проверкой внутри них. Самое главное, не забудьте, что нельзя использовать одинаковые имена итерируемых переменных, когда вы используете вложенные циклы.

```python
n = 0
for i in range(10):
    for j in range(10):
        if i != j:
            n += 1
        else:
            pass

print(n)
```

### Результат

![Меню](https://github.com/GanievaAnastasiia/Software_Engineering/blob/Тема_3/images3/9.png)

### Выводы

Код подсчитывает количество пар (i, j) (в диапазоне от 0 до 9), где i и j не равны друг другу, и выводит результат, равный 90.
Инициализация переменной: переменная n инициализируется значением 0. Использоваться для подсчета количества пар (i, j), где i и j различны.
Внешний цикл for: `for i in range(10):` внешний цикл, который перебирает значения i от 0 до 9.
Внутренний цикл for: `for j in range(10):` внутренний цикл, который также перебирает значения j от 0 до 9 для каждого значения i.
Условная конструкция if: `if i != j:` — проверяет, не равны ли i и j. Если это так (т.е. i и j различны), увеличивает n на 1 (n += 1). 

## Лабораторная работа №10
## Напишите программу с использованием flag, которое будет определять есть ли нечетное число в массиве. В данной задаче flag выступает в роли индикатора встречи нечетного числа в исходном массиве, четных чисел.

```python
list = [42, 16, 8, 67, 31, 9]
flag = False
for i in list:
    if i % 2 != 0:
        flag = True
        break
if flag is True:
    print('В массиве есть нечетное число')
else:
    print('В массиве все числа четые')
```

### Результат

![Меню](https://github.com/GanievaAnastasiia/Software_Engineering/blob/Тема_3/images3/10.png)

### Выводы

Переменная `flag` инициализируется как False. Она будет использоваться для обозначения наличия хотя бы одного нечетного числа в списке.
Цикл for: проходит по каждому элементу i в списке list. Для каждого элемента выполняется проверка, является ли он нечетным `(if i % 2 != 0:)`.
Условие в цикле: если найдено нечетное число, flag устанавливается в True, и с помощью break цикл прерывается. Это означает, что после нахождения первого нечетного числа дальнейшая проверка не нужна.
